VF30

VF30 es un producto de control de acceso profesional diseñado para los requisitos de seguridad de pequeñas y medianas empresas. integra funciones de identificación de huellas dactilares, RFID, PoE TCP / IP, asistencia de tiempo y control de acceso, con una apariencia elegante y una estructura confiable. Con el software de control de acceso independiente - Crosschex, VF30 ofrece control de acceso de grupo para un alto nivel de seguridad. La mejor combinación de Función y Precio.

OA1000

OA1000Pro es un producto insignia de identificación de huellas digitales de Anviz, basado en el sistema operativo Linux, que presenta: CPU de alta velocidad de doble núcleo; gran soporte de memoria; y 1: 10000 velocidad de coincidencia de menos de 0.5 segundos. Aproveche las redes flexibles con una variedad de conexiones de red: TCP / IP, WIFI opcional o módulos de comunicación 3G. El OA1000Pro cuenta con un servidor web incorporado, que permite un acceso conveniente a la configuración del dispositivo y la búsqueda de registros. OA1000Pro con el sistema Anviz Crosschex Cloud, reduce el costo de la configuración del sistema y la aplicación móvil proporciona la gran flexibilidad para la gestión empresarial.

FacePass 7

El producto de reconocimiento facial estrella de FacePass 7 lanzado por Anviz, basado en un nuevo diseño de tecnología infrarroja, utilizando el algoritmo central BioNANO líder en el mundo, con un potente 3050procesador y una plataforma informática de hardware, detección de caras, identificación y eventos en tiempo real, se puede implementar fácilmente. Puede realizar un reconocimiento facial preciso en diversos entornos. CON WIFI RED Y USB
